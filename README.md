
# PROJECT-pill_solution-Frontend
IT 동아리 프로젝트의 프론트엔드 레포지토리입니다. React를 활용하여 진행한 첫 프로젝트이며, 지도 기능은 카카오 맵 API를, 전반적인 UI 디자인은 Bootstrap을 사용하여 구현했습니다.

### 메인 페이지
프로젝트의 첫 화면입니다.
![1](https://user-images.githubusercontent.com/90302922/216917522-6c4b25a6-45f1-4ba7-9b9a-9645491870b3.PNG)

### 상황 선택 페이지
사용자가 현재 겪고 있는 상황(증상)을 클릭하여 선택할 수 있도록 구성한 페이지입니다.
![2](https://user-images.githubusercontent.com/90302922/216917990-231d210d-19a3-4239-ae8e-70bfcb4b58b5.PNG)

### 약 정보 및 치료법 안내 페이지
이전 단계에서 특정 상황을 선택하면, 그에 맞는 약 정보와 치료법이 화면에 출력되도록 구현했습니다.
![3](https://user-images.githubusercontent.com/90302922/216918140-900aaff4-dcf5-4a33-b8da-ba33263ba18f.PNG)

### 상세 정보 외부 링크 연동
치료법 카드 내 버튼을 클릭하면, 해당 질병 및 증상에 대한 상세 정보를 제공하는 외부 사이트로 연결되도록 링크를 삽입했습니다.
![4](https://user-images.githubusercontent.com/90302922/216918657-b69b7870-32dd-44c5-a31c-abeffd806532.PNG)

### 내비게이션 바 및 스크롤 기능
상단 내비게이션 바를 통해 다른 카테고리나 상황으로 손쉽게 이동할 수 있습니다. 콘텐츠 내용이 길어질 경우를 대비해 화면에 스크롤바가 활성화되도록 처리했습니다.
![5](https://user-images.githubusercontent.com/90302922/216918840-9378f2ed-a211-468e-8331-3bdc8dd77db5.PNG)

### 주변 약국 검색 기능
검색창을 통해 사용자의 주변 약국 위치를 파악할 수 있도록 검색 기능을 구현했습니다.
![7](https://user-images.githubusercontent.com/90302922/216919087-be8cae74-718f-478b-8d14-c562909adef0.PNG)

### 프로젝트 구현 영상
프로젝트의 전체적인 구조 설계 및 핵심 기능 구현은 본인이 담당하였으며, 세부적인 UI/UX 디자인은 팀원과의 협업을 통해 완성했습니다.
